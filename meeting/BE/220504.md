- HTTP API 아키텍처 기반으로 API 스펙을 디자인하기로 했다.
- JSON을 직렬화 포맷으로 결정했다.

- 날씨정보 받아오기

```http
GET /weather/{campus_id}
```

- 건의사항 보내기
  // discord api 확인

```http
POST /suggestion/{campus_id}
```

- 캘린더

  - C : 생성

  ```http
  POST /calender/{user_id}
  ```

  - R : 읽기

  ```http
  GET /calender/{user_id}
  ```

  - U : 업데이트

  ```http
  UPDATE /calender/{user_id}
  ```

  - D : 삭제

  ```http
  DELETE /calender/{user_id}
  ```
